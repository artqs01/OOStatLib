Opis szczegółowy

template<Interval DataType, Ratio CalcType = double>
class sl::variance_significance_test< DataType, CalcType >

Klasa implementująca test dla istotności wariancji.

Argumenty szablonowe:

  • DataType: typ danych, z których liczony jest test,
  • CalcType: typ zmiennoprzecinkowy, na którym wykonywane są obliczenia.

Test zwraca true, jeżeli podana wariancja i wariancja podanych danych są istotnie statystycznie równe sobie (prawdziwe jest ), false, jeżeli są różne (należy odrzucić ).
