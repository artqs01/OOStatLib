Opis szczegółowy

template<Interval DataType, Ratio CalcType = double>
class sl::mean_significance_test< DataType, CalcType >

Klasa implementująca test dla istotności średniej.

Argumenty szablonowe:

  • DataType: typ danych, z których liczony jest test,
  • CalcType: typ zmiennoprzecinkowy, na którym wykonywane są obliczenia.

Test zwraca true, jeżeli podana średnia i średnia podanych danych są istotnie statystycznie równe sobie (prawdziwe jest \(H_0\)), false, jeżeli są różne (należy odrzucić \(H_0\)).
